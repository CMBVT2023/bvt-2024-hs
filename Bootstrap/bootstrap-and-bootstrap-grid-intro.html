Notes:

  • Bootstrap is a CSS framework that provides pre-made CSS classes to save time when designing with CSS.
  • Bootstrap mainly covers three things, one is general default styles, two is pre-defined CSS classes, and third is grid layout.
  • General style refers to a package, called reboot, that set cross-browser style default that makes sure all browsers will view a webpage's default values the same. It also handles the default font-family and font sizes. It also handles the default responsive styles of things like tables, images, and figures. Note that all of these can be overwritten
  • Grid layout refers to the containers that house the rows and columns that position all the html elements. By default row uses flexbox to control the positioning of elements.
  • Pre-defined CSS classes / components, bootstrap already has default for styles like alerts, buttons, cards with images, forms. All of these can be imported to allow the quick creation of a website and all of them are highly configurable and you can even nest each component within one another.
  • Bootstrap can also be referred to as a CSS framework or library which means it has pre-made code that can be imported to any CSS project.
  • Always design your website to be usable on a small mobile device, a half computer screen, and a full computer screen. These are the three most common viewport sizes.
  • Always go through the most common device screen sizes when you are creating your website, it is always ideal to make your website adaptable to and usable on the most common screen sizes since those are the most likely to be used when accessing a website.
  • Always design a website starting with the smallest screen size first and then work your way up
  • List the breakpoint modifiers on classes in order of smallest to largest since it will make it easier to read and understand code if every value in in order from least to greatest.
